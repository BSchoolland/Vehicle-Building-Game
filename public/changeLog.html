Changelog

Version 1.1.1 - Bug Fixes - Released on 04-01-2024

Fixes:

  • Fixed bug where level would not end when beating a level you had already completed
  • Fixed bug where the game would not load in certain cases

Changes:

  • Added more bonus objectives to levels in world 2 and 3

Version 1.1.0 - Bonus Objectives and Feature Voting - Released on 03-31-2024

New Features:

  • Added bonus objectives to levels you've already completed
  • Added the feature dashboard, where you can vote on upcoming features, submit feedback, and view this changelog
  • Added new login system, allowing you to save progress across devices

Changes:

  • Replaced coin texture
  • Added "TNT" text to the TNT texture

Fixes:

  • Fixed visual bug when leaving a level before it is fully loaded
  • Fixed objectives not displaying during level tours

Version 1.0.0 - Initial Release - Released on 02-26-2024

New Features:

  • Wrecking Wheels: A unique 2D physics-based game where you build and control vehicles of mahem
  • Explore 33 levels
  • Satisfying destruction and explosions
  • 7 different blocks to build with, including wheels, rocket boosters, and TNT
