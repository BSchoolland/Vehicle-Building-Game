A quick explanation

  • If you have wheels, you can move with 'a' and 'd'
  • You delete blocks by clicking on them and pressing backspace
  • You rotate blocks by clicking on them and pressing R
  • You can create levels in the level editor, you'll have to set up allowed blocks and objectives. Also make sure each level has at least one Building Area Block
  • You can test levels now, using the test level button.
  • If you save a level, you'll get a file in downloads called level.json. If you press load and select that file, it will come back. (this also works for sending levels to your friends, just make sure they save any level they're working on before loading yours because it will overwrite)
