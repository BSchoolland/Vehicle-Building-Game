Vote on What to Add in 1.4.0

Level Editor

Create your own levels, explore levels created by other players, and vote for your favorites. The best levels would have a chance to be added to the main game.

Real Time Strategy Battles

Inspired by the loading screen, a mode where you build an arsenal of vehicles, and are then able to place them on a battlefield to fight against another player's vehicles (or maybe several bot difficulties). The goal would be to destroy the other player's base.

Multiplayer Build Battles

Get a set of random blocks, and build a vehicle with a time limit to battle against another player online. Potentially could have a ranking system and more maps.

Changelog

Version 1.2.1 - CrazyGames Release - Released on 06-26-2024

Changes:

  • Added the game to CrazyGames
  • Added a help button to levels that explains the controls
  • Prevent vehicles from being spawned in with disconnected blocks and display a helpful warning message
  • Remade two world 3 levels (3 and 6) with a similar solution but in a more visually appealing and easier to understand way

Fixes:

  • Fixed bug where users could get stuck in the "rotate device to play" screen
  • Fixed index page sometimes spawning too many enemies

Version 1.2.0 - Sandbox Mode and Major UI Improvements - Released on 05-18-2024

New Features:

  • Added sandbox mode, where you can build with unlimited blocks and no objectives
  • Improved home screen with new buttons and an animated background
  • Updated level select screen for better visuals
  • Remade the build mode UI to be more user-friendly and visually appealing
  • Maps now automatically create dirt blocks below the grass, to make levels look more grounded (pun intended)

Changes:

  • All contraptions must now have at least one block in addition to the seat
  • Made some world 1 bonus objectives easier to complete
  • Buffed fire damage, it deals slightly less damage per second, but lasts much longer

Version 1.1.1 - Bug Fixes - Released on 04-01-2024

Fixes:

  • Fixed bug where level would not end when beating a level you had already completed
  • Fixed bug where the game would not load in certain cases

Changes:

  • Added more bonus objectives to levels in world 2 and 3

Version 1.1.0 - Bonus Objectives and Feature Voting - Released on 03-31-2024

New Features:

  • Added bonus objectives to levels you've already completed
  • Added the feature dashboard, where you can vote on upcoming features, submit feedback, and view this changelog
  • Added new login system, allowing you to save progress across devices

Changes:

  • Replaced coin texture
  • Added "TNT" text to the TNT texture

Fixes:

  • Fixed visual bug when leaving a level before it is fully loaded
  • Fixed objectives not displaying during level tours

Version 1.0.0 - Initial Release - Released on 02-26-2024

New Features:

  • Wrecking Wheels: A unique 2D physics-based game where you build and control vehicles of mahem
  • Explore 33 levels
  • Satisfying destruction and explosions
  • 7 different blocks to build with, including wheels, rocket boosters, and TNT
